How To Run 'Rerobo'

NOTE: "Quotation" marks enclose what you need to type - obviously, you don't actually need to type the quotation marks ;-)

1/ Open Nifty Telnet and select 'New Connection' from the file menu.

2/ Select 'Beetroot' and click 'Connect'.

3/ Login with your username and password.

4/ When you get the command prompt '$' type "su -" and press enter.

5/ You will be asked for a password. Type "xxx#xxx".

6/ You are now the "SuperUser" and can do all sorts of things so be very careful. Each time you do something you will go to a new line and get '#' as a command prompt.

7/ Type 'ps aux | grep java'. This command will show you all the applications running that are using Java.

8/ This should give you a couple of lines, (or more), that look like this:

root   12311   0.5   0.5   732    304    pts/3  R  16:33:27  0:00  grep java
root   12240   0.0   3.0   3000   1888   ?      S  16:18:43  0:35  /disk1/java/bin/..

The first one with "grep java" in it is the one we just did so you can ignore it, it is the second one we are looking for.

9/ The first number in the line is the "process id number", (you can pretty much ignore the rest of the line). You need to 'kill' this process so type "kill 12240" or whatever that id number is.

10/ Now we need to check that this has worked so type "ps aux | grep java" again. You should get only one line ending in 'grep java' this time if the process has successfully been 'killed'. If not, you need to 'really' kill it, so type "kill -9".

11/ Check it again by typing "ps aux | grep java" again.

12/ Repeat steps 10 and 11 as many times as needed until the process is dead.

13/ Now we run 'Rerobo' by typing "/root/rerobo.sh".

14/ If you get a message saying something like 'error connecting to server...' you will need to Timbuktu into the Robomac and restart it.

15/ Open Timbuktu Pro. If the 'New Connection' dialogue box does not open automatically select 'New Connection' from the 'File' menu and select 'robomac' by clicking on it and hitting 'enter'.

16/ You be asked for a password. It is "xxxxxx". A new window will open that will show the desktop of the Robomac.

17/ Quit the Webcam Turbo application then select the 'Desktop' and choose 'Restart' from the 'Special' menu. Be VERY careful not to choose 'Shutdown' as you will then have to go down to the machine and start it by hand.

18/ Open Netscape and check that the RoboCam is working. You should be able to see the commands in the Nifty Telnet window as you move the camera around.

19/ Close the Timbuktu connection.

20/ Log out of Beetroot by typing "exit" and hitting 'enter' once to log out as the SuperUser and then again to log out as yourself.

21/ There - you have done it!
