Setting up a new Mailman list

 

login to polly.va.com.au and su to root.

change directory to /usr/local/mailman/bin/

now its time to set up the new list.

at the prompt type ./newlist

at this point you'll be asked some questions.
1) Enter the name of the list:
2) Enter the email of the person running the list:
3) Initial list-name password:


thats it.. the email you placed in number 2) will be sent the details about the list
including the password. so the first time the owner logs in would be a good time to
change the password.
