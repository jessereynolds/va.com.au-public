Run program before lyris@

This setting allows you extend Lyris command processing with your own program. When mail is sent to lyris@…, your program will first be run, and your program can do any processing it wishes on the message. If your program wants to stop processing of the message, it can delete it.

See How can I extend Lyris with my own programs?.
