Run a program before Phrase

(optional) This setting allows you extend the auto-responder command processing with your own program. When the auto-responder is triggered, your program will first be run, and your program can do any processing it wishes on the message. If your program wants to stop processing of the message, it can delete it.

See How can I extend Lyris with my own programs?.
