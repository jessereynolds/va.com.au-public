Email Address fields

The following fields apply to list postings, are substitute in the values from the currently active posting. They all substitute in email addresses of one type or another.

email.list
The email address for posting to this mailing list. Messages emailed to this address will be contributed to the mailing list.

email.owner
The email address for sending messages to the owner(s) of this mailing list. This is the owner-listname@… address, and all mail sent to this address is distributed to the members of the list who have the setting "receive admin mail: yes" set.

email.unsub
The email address for unsubscribing from this mailing list. This is the "unsubscribe-listname@…" address. The simplest way to unsubscribe from a mailing list is to forward a list posting to this address.

email.purge
The email address for unsubscribing from all mailing lists at this server. This is the "purge@…" address. The simplest way to unsubscribe from all mailing lists is to forward a list posting to this address.
