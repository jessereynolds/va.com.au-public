Step-by-step instructions for Importing Lyris data into Access 97

PROCEDURE SUMMARY: USING ACCESS TO VIEW LYRIS DATABASE FILES

1. Stop Lyris, either by typing "shutdown" at the Lyris console, using the Lyris web Interface or stopping the service.

2. Copy the complete contents of the \lyris\db directory into a backup directory, such as \lyris\dbbackup.

3. Start up Access program. In Access, click on FILE | New Database

4. On GENERAL TAB PAGE, double click on BLANK DATABASE icon.

5. Click on CREATE Button.

6. With TABLE Tab page open, click on FILE | GET EXTERNAL DATA | IMPORT

7. Select File Type = Microsoft FoxPro (*.dbf) for FILE TYPE at bottom of IMPORT page. (This lets you see the dbf files in list)

8. Select and click on the dbf file to GET, such as the Outmail.dbf or Inmail.dbf files.

9. Click on IMPORT button at right hand side of page (wait for a few seconds of importing.)

10. At message = SUCCESSFULLY IMPORTED [FILE NAME], click OK.

11. Click CLOSE button, and see the DATABASE page, with tabs of: Tables, Queries, Forms, Reports, Macros, Modules

12. Double click on listed file you just imported to open that table on the table tab page.
