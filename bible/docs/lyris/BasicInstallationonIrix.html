Basic Installation on Irix

If you downloaded the "gzipped" version for Irix, type this:

    gunzip lyririx.tar.gz
    tar xf lyririx.tar
    cd lyris
    ./install_lyris

If you downloaded the "compressed" version for Irix, type this:

    compress -d lyririx.tar.Z
    tar xf lyririx.tar
    cd lyris
    ./install_lyris
