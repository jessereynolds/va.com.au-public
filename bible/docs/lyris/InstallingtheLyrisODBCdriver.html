Installing the Lyris ODBC driver

To install the Lyris ODBC driver, download it from our web site, at http://www.lyris.com/down

The setup program will ask you to register your serial number. Once you have done this, the setup program will run the Microsoft ODBC setup program. You should choose the "Lyris ODBC Driver" and click "ok". Files will then be copied to your system.

Then, you will be presented with a "data sources" menu. Click the "close" button.

That's it!
