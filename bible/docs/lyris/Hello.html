Hello

(optional) Reference to a document that will be sent via email to new members of this mailing list at the time that they join. Leave Blank if no document should be sent.

Here is a sample hello document, that uses mail merge codes to put the correct information in. You might want to create a server-level hello document with this text in it, so that all lists can use this same hello document, and Lyris will automatically substitute the correct information in.

    Subject: Welcome to $subst('list.name')

    *****************************************************
    PLEASE SAVE THIS WELCOME MESSAGE!
    *****************************************************

    Welcome to $subst('list.name')!

    SUBSCRIBING:
    Send a blank message to: join-$subst('email.list')

    UNSUBSCRIBING:
    Send a message to your personal unsubscribe address: $subst('email.unsub')

    CONTRIBUTING TO THE LIST:
    Send a message to: $subst('email.list')

    WEB INTERFACE: http://$subst('site.domainname')/cgi-bin/lyris.pl?enter=$subst('list.name')

    You can change your subscription email address, unsubscribe, read archives, or change your settings at any time.

    LIST OWNER:
    If you have any questions or problems with your subscription please contact the list owner at: $subst('email.owner')

    Thanks for joining!

    -----------------------------------------------------
