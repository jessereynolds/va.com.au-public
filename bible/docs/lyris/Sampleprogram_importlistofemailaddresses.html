Sample program - import list of email addresses

This sample program demonstrates how to add members directly into a Lyris mailing list. It reads a text file of email addresses (and optionally, names), and creates a member for each email address. The file is named "imp-user.pl" and is available in a subdirectory of your Lyris installation.
