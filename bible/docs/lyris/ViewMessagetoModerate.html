View Message to Moderate

This page allows the List Administrator to read each message that is waiting to be moderated, and decide whether to Approve or Reject the message.

The Moderate Message page has the following elements:

Approval Time
Displays the time when this message will be automatically approved.

From
Displays the email address, and if available, the name of the person who wrote the message.

Subject
Displays the subject line of the message

Body
Displays the body of the message.

Approve Button
Clicking this button will approve this message. These messages will then be distributed to the mailing list. The next message to be moderated, if there is one, will be displayed.

Reject Button
Clicking this button will reject this message. These messages will not be distributed to the mailing list. The next message to be moderated, if there is one, will be displayed.

Edit Button
Clicking this button will let you edit the text of this message. You will be able to edit the header and body separately.

Set automatic approval date and time
This button, and the associated listbox choices for the date/time, let you specify the date and time when this message will be automatically approved by Lyris. You can specify a time up to a 10 minute accuracy.

Should Lyris Rewrite the Date Header:
When this message is approved, this setting determines whether the Date: header on this message will be rewritten to reflect the approval time. The default value from the list settings are inherited for all moderated messages. See Rewrite Moderated Date: Header in the list settings page.

Set Automatic Resubmission
This setting determines whether a message is resubmitted after it has been approved. The purpose of this feature is to have automated periodic mailings, such as a "frequently asked questions" document, sent to your mailing list automatically. By default, messages are not resubmitted. If you do choose to use this resubmission feature, you can specify in how many days the message will be automatically approved.

Go Back to Moderated Menu
Clicking this button returns you to the first Moderated Messages page..

Go to list admin menu
Clicking this button returns you to the List Admin menu.
