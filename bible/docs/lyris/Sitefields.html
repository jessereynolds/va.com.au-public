Site fields

The following fields apply to list postings, are substitute in the values from the currently active site (i.e.: the site which hosts this mailing list).

site.name
The short name of this site.

site.domainname
The host name this site uses, example: "lyris.shelby.com"

site.country
The country this site is located in.

site.city
The city this site is located in.

site.urlowner
A URL to the home page of this site's owner.

site.ownergif
A URL to a GIF/JPEG graphic logo for this site owner.

site.urllyris
A URL to the Lyris home page for this site (if any)

site.mailaddr
The postal mailing address of this site.

site.desc
A short description of this site.

site.deflang
The default language used at this site.

site.mainadmin
The name of the main administrator defined for this site.

site.techsupp
The name of the technical support contact defined for this site.

site.desc
A longer description of this site.
