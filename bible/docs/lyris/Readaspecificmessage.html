Read a specific message

This page permits the user to read the complete text of email messages, and respond to them if the user wishes to.

The Read Message page has the following elements:

Message Set
This line of text informs the user what message is currently being displayed, relative to the full message set. The message set this line is referring to is the message set that was displayed on the Read and Search Messages page when the user clicked the View Message button.

Message Information
The author's name and email address, subject line and message body are all displayed here. References to World-Wide-Web URLs are identified and hot-linked. If the user clicks on a link in a message, the web browser will load the designated link.

Reply To This Message
The user can click on this button to bring up the Create a New Message page, in order to post a follow-up message to the currently displayed message. The subject line will be Re: <current subject>. If the Include Message in Reply checkbox is selected, the current message will be quoted and pre-included in the New Message Window. The user will be able to edit the quoted section as they wish.

Create New Message
The user can click on this button to bring up the Create a New Message page. This button is for creating a new message, with a new subject line. See Create a New Message.

Back to the Read Menu
Clicking on this button brings the user back to the Read and Search Messages page.

Back to the Main Menu
Clicking on this button brings the user back to the Menu of Member Choices page.

Delete Message
If you are a list administrator, a button will appear titled "delete message". By clicking on this button you will cause the current message to be deleted.
