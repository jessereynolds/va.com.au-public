Command-Listname and Listname-Command

This is an alternate address for sending Lyris commands, and for most commands, is the simplest way to execute them. Any mail sent to these addresses executes the command in the address. The text of the message sent is not relevant in the command.

For example, to subscribe to a mailing list called "jazztalk", people can send mail to "jazztalk-subscribe" ("subscribe-jazztalk" also works) Listname-Command@… and Command-Listname@ are equivalent. Both will work, and both mean the same thing.

The commands subscribe (and aliases), unsubscribe (and aliases), purge, mail, digest, index, nomail, ack, noack, unhold, query, confirm, review, stats, hello, goodbye are available in this way.

See the unsubscribe and purge section for special information about those commands.

For more information, see Unsubscribe
