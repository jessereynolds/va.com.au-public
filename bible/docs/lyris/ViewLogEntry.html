View Log Entry

After clicking the View Log Entry button from the first log screen, you will be brought to a full page displaying the complete log entry.

The View Log Entry page has the following elements:

Created
The date the log entry was created in the system

Source
The functional section within Lyris that caused this log entry.

User Name
If applicable, the user name which was active during this log entry, or which caused this log entry.

Referrer
If applicable, the object which caused the log entry.

Title
The full text title of this log entry.

Site
If applicable, the Site for which this log entry applies. If the log entry applies to all sites, then the Site is local.
