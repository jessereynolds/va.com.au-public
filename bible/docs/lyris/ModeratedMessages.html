Moderated Messages

The Moderated Messages page allows you to read, approve or reject messages which need list administrator approval before being sent to the mailing list for distribution. If the mailing list is not using some form of list moderating, this menu will always display the phrase "No Messages are waiting to be moderated".

For more information about setting up a moderated list, see Moderated, or the FAQ at How can I make a moderated list?. You also have the option of approving moderated messages via email. See Moderate for more information about this.
