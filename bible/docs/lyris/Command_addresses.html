Command@ addresses

An alternate address for sending Lyris commands that do not apply to a particular list.

The command@ address does the command for which it is named.

The commands purge, unhold, help, info that are available in this way. For example, sending mail to purge@clio.lyris.net will unsubscribe you from any mailing lists you are on at the Lyris server running at clio.lyris.net.
