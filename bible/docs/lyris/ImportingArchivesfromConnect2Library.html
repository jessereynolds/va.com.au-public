Importing Archives from Connect2Library

Lyris comes with a program, named "imp-c2l.pl" to import mailing list archives from the DOS-based list server program named "Connect2Library".

Each message is stored one after another in a file, with a long list of dashes separating each messages. You can import as many mail files as you have.

The messages will be full text indexed and appear in Lyris as old messages, as if they had been created in Lyris. They can be searched from the Lyris Web Interface as well as by email.

You run this program from the command line. To run it you need to specify the mailing list name, and the names of the text files you want to import. For example:

    perl imp-c2l.pl jazztalk 02-1997.msg
    perl imp-c2l.pl jazztalk 0297.msg 0397.msg 0497.msg
    perl imp-c2l.pl jazztalk *.msg

You can specify as many text files to import as you like.

This program is available in the "extra" section of our FTP site, at:

    ftp://ftp.lyris.com/extra/

Once you download imp-c2l.pl, you should copy it to your CGI directory, which is the same directory where "lyris.pl" is located on your web server.
