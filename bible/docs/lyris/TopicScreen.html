Topic Screen

The Topic screen is the Web page that displays after you have selected a topic from the Lyris Home Page. This page displays all the mailing lists for a given topic.

Topic Logo
The graphic for the selected Topic is displayed at the top of the screen. If there is no graphic defined for that topic, the text name of the topic is displayed instead.

Page Title
The title of this page tells you that these are the mailing lists for this topic.

Mailing lists
Every mailing list for this topic is displayed here. If a mailing list has a graphic defined (as is the case in this example), then the graphic is displayed instead of the mailing list name in text.

Home button
Click the home button to go back to the Lyris Home Page. The home button appears on most Lyris Web Interface pages.
