Allow Joining by Email

This setting determines whether subscription requests by email will be accepted. By default, members can join a mailing list using the Lyris web interface, or by sending a "subscribe" command to Lyris. For more information about subscribing, see Subscribing.

If you want to require people to join using the Lyris web interface, you can change this setting so that requests to join by email are rejected. For more information, see Ask Subscribers for Name or Ask Subscribers for Password.
