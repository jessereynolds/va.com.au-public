First People Page

The People Menu page has the following elements:

Person List
The currently existing People are listed here. Each Person is listed in one line of the list box.

Edit Person
Clicking this button will bring you to an Edit Person page, with all the same fields as the Create Person page. The Person to be edited will be the Person currently selected in the Person List.

Delete Person
Clicking this button delete the Person currently selected in the Person List.

Create Person
Clicking this button will bring you to the Create Person Page.
