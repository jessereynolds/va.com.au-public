Server Documents

The "documents" choice lets you create text documents for use inside Lyris. For instance, you can define the contact information for your technical support department, and reference it every time it's appropriate (for each mailing list, for instance). You can also have multiple language version of the same document - Lyris will display the right one.

Documents created within the Server Documents menu will be visible on all Sites, but will only be editable from the Server Documents menu. To create documents that are only visible within one site, create the Document using the Site Documents menu.

If you click on the Server Documents button on the Server Admin Menu, a page will be displayed where you can choose to either Edit or Delete an existing Document, or create a new one.
