Testing the Lyris ODBC Driver

Once you have configured your Lyris data source, you can test it.

These instructions apply to Access 97.

* start Access 97

* load the "Northwind" sample database, or any other existing database.

* Choose the menu "File/get external data/link tables"

* in the "files of type" choice, choose "ODBC databases"

* click on "Machine data source" and choose "Lyris"

* When the "choose tables" menu pops up, choose "members"

* When the "select unique record identifier" menu pops up, choose "MemberID"

* that's it! You can now browse the "members" table from Microsoft access.
