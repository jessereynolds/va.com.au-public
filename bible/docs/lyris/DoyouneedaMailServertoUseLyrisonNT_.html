Do you need a Mail Server to Use Lyris on NT?

> Do I still require another mail server with Lyris
> like MailSite, NTmail, SLmailNT, etc. or does
> Lyris do everything?

Lyris will do everything relating to mail -- you do not need another mail server. It has its own mail server built in.

Note that if you already have a mail server and want Lyris to work on the same machine, Lyris can be made to coexist with another mail server. See Having Sendmail coexist with Lyris .
