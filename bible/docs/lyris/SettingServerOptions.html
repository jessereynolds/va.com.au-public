Setting Server Options

The following command line parameters let you set Lyris configuration options. All these parameters can be changed from the "Server Configuration" Web Interface page. The command line versions are included for convenience. For more information, see Server Config.

activate
Set Lyris activation code Number to that specified on the command line.

dns
Set Domain Name Server to the TCP/IP address specified on the command line. DNS Server.

email
Sets the email address of the administrator of this Lyris server.

forward
Set mail forwarding rule to "ip-address port". See Forward Mail.

lcp
Add the TCP/IP address specified on the command line to the list of allowed LCP hosts.

lcpport
Sets the TCP/IP port that Lyris accepts incoming LCP connections on. The default is 2020. See LCP Port.

lcppass
Sets the password that the LCP protocol uses to log into this Lyris server.

Note: you must change the password= line of the lyris.plc file to match this password, if you do change it.

tcpip
Specifies the TCP/IP addresses that this Lyris server is allowed to use. See Multiple TCP/IP addresses.

gettcpip
Specifies the TCP/IP address that you should use when you register your serial number to obtain an activation code.

url
Set URL that goes to Lyris Web Interface. See Lyris URL.

version
Displays the version of Lyris that are running.

help
Displays the command line help.

smtpport
Sets the TCP/IP port that Lyris accepts incoming SMTP connections on. The default is 25. See SMTP Port.

nntpport
Sets the TCP/IP port that Lyris accepts incoming NNTP connections on. The default is 119. See Newsgroup Port.
