Topics First Page

The Topics Menu page has the following elements:

Topic List
The currently existing Topics are listed here. Each Topic is listed in one line of the list box.

Edit Topic
Clicking this button will bring you to an Edit Topic page, with all the same fields as the Create Topic page. The Topic to be edited will be the Topic currently selected in the Topic List.

Delete Topic
Clicking this button deletes the Topic currently selected in the Topic List.

Create Topic
Clicking this button will bring you to the Create Topic Page.
