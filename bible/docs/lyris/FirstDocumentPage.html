First Document Page

The Documents Menu has the following elements:

Document List
The currently existing Server Documents are listed here. Each document is listed in one line of the list box. First, the type of document is given in all capital letters, followed by the title of the document, with the language of the document after it, in parentheses. You can create multiple language versions of the same document.

Edit Document
Clicking this button will bring you to an Edit Document page, with all the same fields as the Create Document page. The document to be edited will be the document currently selected in the Document List.

Delete Document
Clicking this button delete the document currently selected in the Document List.

Create Document
Clicking this button will bring you to the Create Document Page.
