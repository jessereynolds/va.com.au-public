The Listname-Request Address

Email Address: Listname-request@

Purpose: an alternative to Lyris@, this email address will accept all Lyris commands, except that the listname does not need to be specified in the command (but it will work fine if it is).

Example address: jazztalk-request@clio.lyris.net. Example command: "subscribe"
