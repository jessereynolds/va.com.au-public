Which

The which command allows an administrator to determine which mailing lists a particular person belongs to. The syntax of the which command is:

    which <email address> …

You can specify as many email addresses on the which command line as you like. For example, the follow commands are valid:

    which bob@somewhere.com
    which jane@somewhere.com albert@elsewhere.com

The which command is only available to administrators. You must issue a "login" command before using the send command. For more information about login, see Login.

Note that if you do not first login as an administrator, the which command will tell you which lists you are on. See Determine Membership Settings.
