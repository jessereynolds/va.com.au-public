Server Admin Menu

The Server Admin Menu has the following elements:

Sites
One Lyris Server can have as many sites as you wish. Each Site will have its own Lyris Home Page, as well as its own topics and mailing lists. Once you create a site, you can assign people to be Site Administrators, so that they have complete control over their own site, but act independently of the other sites.

For example, if you are a company with multiple departments, you could create a different site for each department. Then, someone in each of those departments could be made a Site Administrator, and they would be responsible for the mailing lists on their Site. This is an excellent way to delegate responsibilities.

Another example: if you are an Internet Service Provider, you probably want to have the least day-to-day involvement with Lyris, and want your paying customers to be responsible for their own mailing lists. If you give each customer their own Site, they will have total control over that site. Each customer will act independently of the others, and will not be able to damage what goes on in other sites. See Create Site.

Server Config
Clicking this button brings you to the Server Configuration page. This page stores several settings that apply to the entire Lyris server, such as your Lyris Serial number, and DNS server. See Server Config.

Server documents
Clicking this button brings you to the Documents page. Documents are used in various places within Lyris, such as the "Welcome to the mailing list" document, or a document describing your organization. Documents created in the Server Admin menu are visible in all sites. On the other hand, Documents created in the Site Admin menu are visible only in the site in which they are created. See Server Documents.

Action Phrases
Clicking on this button brings you to the Action Phrase page. Action Phrases tell Lyris to look for certain words and respond by mailing the document you specify. See Server Action Phrases.

Auto-responders
Clicking on this button brings you to the Auto-responder page. An autoresponder is an email address that always responds by sending a document the person who sent email to it. See Auto-Responders.

People
Clicking on this button brings you to the People page. You can create people on your server who hold a special role. This allows other people to be Server or List Administrators, and can also be used to hold their contact information. See Server People.

Incoming Mail
Clicking on this button brings you to the view Incoming Mail Page. You can view all the mail that Lyris has received from this page. See Incoming Mail .

Outgoing Mail
Clicking on this button brings you to the view Outgoing Mail Page. You can view all the mail that Lyris has sent or is sending from this page. See Outgoing Mail.

Log
Clicking on this button brings you to the Log Page. You can view the log from there that displays various operating information. See Server Admin Log.

Shutdown Server
Clicking on this button brings you to a confirmation page that asks you to confirm that they really want to stop Lyris and shutdown the server. Once you accept the confirmation, the Lyris Server is shut down. All mail activity stops and mail cannot be sent to or from the lists until the server is restarted. Also at this point, the Web Interface will no longer work, since the Lyris server is no longer running. See Shutdown Server.
