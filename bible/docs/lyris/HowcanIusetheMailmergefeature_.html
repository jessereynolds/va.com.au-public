How can I use the Mail merge feature?

Lyris allows you to place fields in an email message which indicate that member-specific information should be placed there. For instance, you can have each user's name included in the message body, as "Dear so and so…".

To use this feature, simply include the Lyris field tag where you want the text replaced. This can be in the permanent header or footer definitions for the list, or anywhere in the body of your messages.

As of Lyris version 2.5, there are many mail merge fields available. The mail merge fields are not case-sensitive.
