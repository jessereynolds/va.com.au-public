How to start the Lyris server

To start the Lyris server, click on the Lyris icon that the install program created for you. If you are running Unix, you should run the program titled "lyris" in the directory where you installed Lyris.

You will see the words "Lyris Server Console" once Lyris is running.

In general, because mail could be coming in to Lyris at any time, it is wise to have Lyris running all the time.
