Your Email Address

To change your email address, use the email= option The syntax for this command is:

    set listname email=[new-email-address]

For example, if you are currently subscribed to jazztalk with the address bob@acme.com and want to be subscribed as bob@widget.com you would send the command:

    set jazztalk email=bob@widget.com

Note: the text after the email= cannot have any spaces in it.
