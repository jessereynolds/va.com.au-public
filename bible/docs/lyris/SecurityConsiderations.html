Security Considerations

When running any server program on the Internet, security must be considered in order to prevent unwanted use of the server. A balance is usually struck between ease of use, and effectiveness of the security precautions. Lyris was built with security considerations at its core, and presents a sophisticated rights, roles and securities model.
