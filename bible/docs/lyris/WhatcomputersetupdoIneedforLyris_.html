What computer setup do I need for Lyris?

To use Lyris, you will need:

1) A computer
2) A TCP/IP Internet connection (a full time connection is recommended if you want users on the Internet to be able to use the Lyris Web Interface)
3) A Web browser (we recommend Netscape or Internet Explorer)
4) A Web server (most people run the web server on the same machine as Lyris, but this is not required)
5) 10 megabytes of free disk space.
6) 5 megabytes of free RAM after the operating system loads
