Site Admin Menu

The Site Admin Menu has the following elements:

Site Info
Clicking this button brings you to the Site Info page. From this page, the administrator can edit all the information about their site. The Site Info page has the same fields and choices as the Server Admin Create Site page.

Topics
Clicking this button brings you to the Topics page. From this page, you can create, edit or delete Topics. Topics are logical categories that are used to organize mailing lists. Every mailing list is organized into a topic, and multiple mailing lists can be organized inside the same topic. See Create Topic Page.

Mailing lists
Clicking this button brings you to the Mailing Lists page. From this page, you can create, edit or delete Mailing Lists. See Mailing Lists.

Documents
Clicking this button brings you to the first Documents page. The menus and choices are identical to the Server Admin Documents pages, except that only the Documents created by this Site are displayed. See Server Documents.

Action Phrases
Clicking this button brings you to the first Action Phrases page. The menus and choices are identical to the Server Admin Action Phrases page, except that only the Action Phrases created by this Site are displayed. See Site Action Phrases.

Site Auto-Responders
Clicking this button brings you to the first Auto-responders page. The menus and choices are identical to the Server Admin Auto-responders page, except that only the Auto-responders created by this Site are displayed. See Auto-Responder.

People
Clicking this button brings you to the first People page. The menus and choices are identical to the Server Admin People page, except that only the People created by this Site are displayed. See Site People.

Log
Clicking this button brings you to the first Log page. This page is identical in all ways to the Server Admin Log page. Server Admin Log.
