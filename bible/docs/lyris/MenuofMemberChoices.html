Menu of Member Choices

From this page, a member can access all the major functions available to them. If the user entered the list as a visitor, only the Read Messages, About This List and Search selections will be available.

The Menu of Member Choices has the following elements:

Read Messages
Pressing this button brings the user to the Read and Search Messages page. See Read and Search Messages.

Create a New Message
Pressing this button brings the user to the Create a New Message page. See Create a New Message.

Your Settings
Pressing this button brings the user to a page with all their member settings. These are the settings the user chose when joining the mailing list. The screen looks exactly the same as the Join Mailing List page, except that the member is now editing their settings, rather than joining. This page is quite useful if, for instance, the user's email address has changed. You can change your email address using this page. For more information, see Edit Membership settings.

Leave (unsubscribe)
This button is used to leave a discussion group. See Leave.

About This List
Pressing this button brings the user to a page of general information about the mailing list.

Search
The user can enter words to search for and press the Search button. The Read and Search Messages page will be brought up, along with a list of messages which match the user's specified search. To search for messages using email, use the search command. See Read and Search Messages.
