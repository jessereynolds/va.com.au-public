Comment

You can change your "comment" to be anything you like. The "comment" information is not used by Lyris for any purposes: it is merely a placeholder for any information you care to place in there. For example, you might use it to store the city you live in.

To set the comment, use the comment= keyword, as in:

    set jazztalk comment=Washington

Note: the text after the comment= cannot have any spaces in it.
