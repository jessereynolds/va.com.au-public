TCP/IP Addresses of Lyris Users

user_ip=

If specified, this defines a mask for TCP/IP addresses that are allowed to access the user menu. If undefined, all users are allowed to see the user menu.

For example, if you have a Lyris Server which is internal to your company, and outsides should not be able to access it, then you can set user_ip to be a mask for the TCP/IP addresses in your company. See the "admin_ip" parameter for more information about masks.

If a TCP/IP address matches the "admin_ip" parameter, it automatically has access to the user menus as well.

You can specify as many user_ip address as you like. The following example would allow access to TCP/IP addresses beginning with 193.158.17.* and 198.15.* as well as to the specific address 194.204.17.2

    user_ip=193.158.17.
    user_ip=194.204.17.2
    user_ip=198.15.
