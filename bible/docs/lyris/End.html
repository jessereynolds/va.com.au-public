The "end" command tells Lyris that there are no more email commands to process in this message. Lyris will stop processing your message when it sees the "end" command. The "end" command is optional. If you do not include it, Lyris will automatically scan to the bottom of your message for commands.

Here is an example:

    subscribe jazztalk
    set jazztalk digest
    end
