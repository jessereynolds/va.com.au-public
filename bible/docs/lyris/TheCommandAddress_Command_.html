The Command Address: Command@

Email Address: Command@

Purpose: an alternate address for sending Lyris commands. The command@ address does the command for which it is named. The commands purge, unhold, help, info, which are available in this way.

For example, sending mail to purge@clio.lyris.net will unsubscribe you from any mailing lists you are on. For more information, see Lyris Email Commands.
