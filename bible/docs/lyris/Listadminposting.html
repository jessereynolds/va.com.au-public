List admin posting

Only members that are list administrators can post. Messages from others are automatically rejected. This is the "admin send" feature. Authentication is performed on the From: address. This is a basic form of security that is fairly easily to bypass. For mailing lists where security is important, we recommend augmenting this security option with another security method, such as moderating or password protected posting.
