User ID

You can change your "user id" to be anything you like. The "user id" information is not used by Lyris for any purposes: it is merely a placeholder for any information you care to place in there. For example, you might use it to store your social security number, or perhaps the Member ID from some other database.

To set the user id, use the userid= keyword, as in:

    set jazztalk userid=045-76-1230

Note: the text after the userid= cannot have any spaces in it.
