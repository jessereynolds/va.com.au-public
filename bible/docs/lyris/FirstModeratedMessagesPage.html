First Moderated Messages Page

The Moderated Messages Menu has the following elements:

Messages to be Moderated
Lists the subject line of each message waiting to be moderated. You can select one or several messages at a time.

Approve Button
Clicking this button will approve all the Messages to be Moderated that are currently selected. These messages will then be distributed to the mailing list.

Reject Button
Clicking this button will reject all the Messages to be Moderated that are currently selected. These messages will not be distributed to the mailing list.

View Message
Clicking this button displays the View Message to Moderate page.

Go to list admin menu
Clicking this button returns you to the List Admin menu.
