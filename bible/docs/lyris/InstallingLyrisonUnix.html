Installing Lyris on Unix

This document explains how to install Lyris on Unix, how to install Lyris by hand, how to install the Lyris Web Interface on a separate machine, and how to make Lyris and Sendmail coexist.

First read Having Sendmail coexist with Lyris and decide what technique you want to use to have sendmail and Lyris coexist on your machine. If you are using Qmail, read Having Qmail coexist with Lyris.

Next, install Lyris by running the Lyris installation program, "install_lyris".

If you have previously installed Lyris, you should still run the "install_lyris" program. It will ask you if you are upgrading an existing copy of Lyris, and if you answer yes, will automatically upgrade your Lyris installation for you.
