Listname-request@ commands

When commands are sent to this address, all the same commands work, but the listname is no longer needed in the command. For instance, one command to subscribe to a mailing list is:

    subscribe listname

that command syntax will work for any of the List Server addresses. However, if the command is sent to listname-request, such as jazz-request@minerva.acme.com, this command is enough, since the list name is clear from the mailing address:

    subscribe

Note that all List Server Address syntaxes work fine when sent to the listname-request address.
