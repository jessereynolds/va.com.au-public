First Auto-Responder Page

The Auto Responder Menu has the following elements:

Auto-Responder List
The currently existing Auto-Responders are listed here. Each Auto-Responder is listed in one line of the list box.

Edit Auto-Responder
Clicking this button will bring you to an Edit Auto-Responder page, with all the same fields as the Create Auto-Responder page. The Auto-Responder to be edited will be the Auto-Responder currently selected in the Auto-Responder Phrase List.

Delete Auto-Responder
Clicking this button deletes the Auto-Responder currently selected in the Auto-Responder List.

Create Auto-Responder
Clicking this button will bring you to the Create Auto-Responder Page.
