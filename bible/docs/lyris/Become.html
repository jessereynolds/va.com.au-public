Become

The become commands allows administrators to become someone else, assuming their email address and name, and issue Lyris commands on their behalf. For instance, an administrator can become another person, and issue the "subscribe" command for them.

There are two possible syntaxes for the become command:

    become email-address
    become email-address Full Name

The first format only specifies an email address. The name is then left blank.

The second format species both the email address and name of the person you are becoming. All words after the email-address are understood as being the name.

The become command is only available to administrators. You must issue a "login" command before using the become command. For more information about login, see Login.
