Lyris Face Graphic

html_lyris_face=
html_lyris_face_text=

You can redefine the HTML that is used to display the small Lyris Face Graphic in the Lyris Web interface. For example, to have nothing appear in that space, use the following definition:

    html_lyris_face=<!-- -->

The html_lyris_face_text option is displayed when "graphics off" is enabled in the Lyris Web interface.
