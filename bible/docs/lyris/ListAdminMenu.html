List Admin Menu

The List Admin Menu has the following elements:

List Info
Clicking this button brings you to the Edit Mailing List page. This page is identical to the Site Admin Create Mailing List page, except that you are editing, rather than creating the mailing list settings.

Members
Clicking this button brings you to the Members page, where you can delete, edit or add members to the mailing list. From this page, you can also edit and create List Administrators.

Moderate
Clicking this button brings you to the Moderate page. From this page you can read, approve or reject moderated messages.

List Documents
Clicking this button brings you to the first Documents page. The menus and choices are identical to the Server Admin Documents pages, except that only the Documents created by this Mailing List are displayed. See List Documents.

List Action Phrases
Clicking this button brings you to the first Action Phrases page. The menus and choices are identical to the Server Admin Action Phrases page, except that only the Action Phrases created by this Mailing List are displayed. See List Action Phrases.

List Auto-Responders
Clicking this button brings you to the first Auto-responders page. The menus and choices are identical to the Server Admin Auto-responders page, except that only the Auto-responders created by this Mailing List are displayed. See List Auto Responders.

Incoming Mail
This button lets you see mail that Lyris has received which pertains to your mailing list. It is the same page as the Server Admin Incoming Mail page, but with the power to see only your list's mail

Outgoing Mail
This button lets you see mail that Lyris has sent or is sending which pertains to your mailing list. It is the same page as the Server Admin Outgoing Mail page, but with the power to see only your list's mail.

Enter List
Clicking this button brings you to the "enter mailing list" page, where you can join, enter the mailing or visit, just as a regular user of your list would. See Enter Mailing List.
