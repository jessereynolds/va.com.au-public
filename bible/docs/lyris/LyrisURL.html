Lyris URL

The Web URL that leads to the Lyris Web Interface home page on this server. Example:

    http://clio.lyris.net/cgi-bin/lyris.pl

When Lyris sends out email addresses with URLs to the Lyris Web interface, it uses this setting as the basis for building each page-specific URL.
