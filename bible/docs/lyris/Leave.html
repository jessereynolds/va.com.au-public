Leave

Pressing the Leave button from the Menu of Member Choices brings the user to a confirmation page that asks if the user is absolutely certain that they want to stop being a member of the mailing list.

If they press the Yes button at that point, the user will be removed, and will no longer be a member of the mailing list.

To leave a mailing list using email, use the unsubscribe command.
