Backwards Compatibility of Subscribing Methods

Several public domain list server software programs use this format of subscribing, and Lyris supports this convention for backward compatibility. Lyris will also accept email at all the common alternate list server addresses (such as majordomo@yourserver.com) and act correctly on this email, as if it were sent to the lyris@yourserver.com address. Lyris also supports the majordomo-standard listname-request@yourserver.com address.
