How to change the CD in the back-up server:

1/ The machine will prompt you for a new CD and tell what to number it. The message will say something like 'Please choose a new CD. It will be called Good Old Days 2 ###'

2/ Take a new CD and number it appropriately and don't forget to put the date on it. There should be a texta by the machine. Check the VACD number of the previous CD and label the new one as the next number in line.

3/ Remember to write all information on both the CD and its cover ­ on the front and on both spines.

4/ Pop it into the machine and push the tray in.

5/ You may need to wait a few moments for the machine to erase the new CD, then click on the 'Proceed' button.

6/ After an hour or so, check the machine again. It may ask you to put the previous CD back in.
