name  // tCR's 'Weaver' remix of Lino's 'Auger Well'  
creator  // Ed Wilson 
project  //REMIX 
We liked the original Lino track, and the elements provided on the noise website - especially the ambience sample.


We created two mixes. The first we tried to use as many of the original
samples as possible keeping a similar style to the original track while
adding a few of our own trademarks. The second was a trance mix, using only
the vocal and ambience samples. This version reflects more of our
trancey/techno style.

We added new drum riff samples, a new lead and various effects.

Tech Stuffs//Cubase VST, Yamaha A3000 Sampler, Nord Lead 2, Yamaha AN1x, Roland Xp30 and
Korg Trinity Rack.

Taking the elements we liked from the original track, we cut them up and
processed them with our sampler. Using this technique we speed up the drum
riff slightly and added our own drum samples. We then sequenced our own
original elements.

Fav remixes//
Exchange - Precious Heart
LSG - Into Deep Remixes
Lots of remixes produced by Deep Dish.

Fav Locals// Frontside, Narcotik, and Powderfinger.

Tips//
Choose a track which you like, and has elements which you think has
potential for creating new sounds. When using loops from other sources, try
to cut them up as much as possible and add your own ideas. The mixdown to
the final track is a very important stage. You can have create ideas but
the end result can sound terrible without proper recording techniques. Look
into mastering and recording.


Ed Wilson is a finalist of the noise Remix project. tCR's 'Weaver' remix of Lino's 'Auger Well' can be heard on triple j through out the month of October.

You can check it out now simply by downloading the track below.
